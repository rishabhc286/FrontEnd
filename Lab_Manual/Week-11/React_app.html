How to Setup a React Project

1. Using Vite (Faster)

Open terminal and run:

npm create vite@latest my-react-app

Then select:

  • Framework: React
  • Variant: JavaScript

Next run:

cd my-react-app
npm install
npm run dev

Your app will run in the browser on a local address.

2. Using Create React App

Run the following command in terminal:

npx create-react-app my-react-app

Then go inside the folder:

cd my-react-app

Start the project:

npm start

React app will start on http://localhost:3000/
