JSX Example with Variables and Functions

In React we can write JavaScript directly inside JSX. Here is a small example where:

  • We store a name in a variable
  • We use a function to generate greeting
  • JSX shows the output on the screen

App.jsx

function App() {

  const studentName = "Garv";

  function greeting() {
    return "Welcome " + studentName + " to React!";
  }

  return (
    

      
{greeting()}

    

  );
}

export default App;

main.jsx

import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App.jsx";

ReactDOM.createRoot(
  document.getElementById("root")
).render(
  
);

This will show a dynamic message like:

Welcome Garv to React!
